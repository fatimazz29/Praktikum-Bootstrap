Life Goes On

Pengalaman Kuliah Pemrograman Web

Pada hari Rabu, 4 September 2024, saya menghadiri perkuliahan Pemrograman Web di kampus yang diadakan secara offline. Pada sesi tersebut, setiap mahasiswa diminta menjelaskan berbagai macam tag HTML secara bergantian berdasarkan urutan absen. Sebelum giliran saya maju, dosen tiba-tiba memberikan soal mendadak. Kami diminta membuat perkenalan diri di Visual Studio Code (VSCode) menggunakan tag HTML tertentu yang tidak ditampilkan. Hanya lima mahasiswa tercepat yang akan mendapatkan nilai, dan beruntungnya, karena saya memperhatikan penjelasan teman-teman sebelumnya dan sudah belajar sedikit tentang HTML, saya berhasil menyelesaikan tugas dengan menjadi yang kedua tercepat.

Setelah menyelesaikan soal, perkuliahan kembali dilanjutkan. Saat giliran saya tiba, saya menjelaskan tentang tag label di HTML. Kami semua melanjutkan presentasi hingga sesi perkuliahan selesai. Setelah kelas berakhir, kami tidak langsung pulang. Sebelum meninggalkan kampus, kami sempat berfoto bersama sebagai kenang-kenangan, lalu saya pun langsung pulang.
